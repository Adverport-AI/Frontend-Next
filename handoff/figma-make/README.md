# Figma Make Handoff

Figma Make exportlarini bu klasore koy.

Onerilen kullanim:
- Son zip: `.handoff/figma-make/current.zip`
- Acilmis klasor: `.handoff/figma-make/current/`

Calisma sekli:
- Exportu ayni adla guncelle.
- Sonra bana `figma make export yenilendi` diye haber ver.
- Ben farklari inceleyip Next.js koduna tasirim.

Not:
- Bu klasordeki export dosyalari ve acilmis klasorler `.gitignore` ile dislanir.
- Sadece bu README takip edilir.
